<section id="標記規則" class="level2">
<h2>標記規則</h2>
<p>基於能與國際學者及各語料庫之間參考互惠的考量，本語料庫所採用的語料標記基本上遵循 <a href="http://www.eva.mpg.de/lingua/resources/glossing-rules.php">Leipzig Glossing Rules</a> 的規則。其中，為因應臺灣南島語的特性，本語料庫對於Rule 6, 7 和10的標記方式做了調整，包括了：</p>
<ol type="1">
<li>第七條 Inherent categories 不使用。</li>
<li>第六條 Non-overt elements 不採用 [ ] 或 Ø 而改用: <em>'akoy</em> (賽夏語) 標為 <span class="sc">af</span>.many。鄒語因焦點構詞較為特殊，non-overt elements如<em>bonU</em> (吃) 改標為 eat.AF，請見5之說明。</li>
<li>第十條 Reduplication 不使用 ~, 目前對於三類的 reduplication 分別採三種處理方式：
<ol type="A">
<li>Ca reduplication：如果重疊是詞首子音重疊加上母音a，則改標成 Ca， 如：ha-hila (賽夏語) 即標為 Ca-sun/Ca重疊-日。</li>
<li>前綴型：如果重疊是前綴型，則標為 <span class="sc">red</span>-，如：m-li-lizaq (噶瑪蘭)，標為：<span class="sc">af</span>-<span class="sc">red</span>-happy/主焦-重疊-高興。</li>
<li>中綴型：如果只有部分中綴重疊而不是所有語幹重疊，則改用&lt;&gt;符號，如：sa-ru&lt;mi'a&gt;mi'ad (阿美語) 標為 sa-&lt;<span class="sc">red</span>&gt;day。</li>
</ol></li>
<li>本語料庫採用 <span class="sc">af</span>, <span class="sc">pf</span>, <span class="sc">rf</span> (<span class="sc">if</span>), <span class="sc">lf</span>等標示, 分別表 Agent Focus, Patient Focus, Referential Focus (Instrumental Focus), 及 Locative Focus。</li>
<li>鄒語焦點的構詞與其他台灣南島語言不同，主事與受事焦點的區別主要在動詞後綴，因此本語料庫把鄒語焦點標記在動詞詞綴之後，例如bonU/ana (吃)標為eat.AF/eat.PF，boacU/eoeca (咬住) 標為bite.AF/bite.PF。少數動詞主事焦點類似其他南島語有-m-, m-形式，比較其受事焦點變化，大多仍伴隨字尾變化，例如tmoecU/teoca(砍)，matvo'ho/patvo'ha (說得出口)，mooteo/totea (等待)。這些例子我們仍維持標記為hack.AF/hack.PF, dare.to.say.AF/dare.to.say.PF, wait.AF/wait.PF。</li>
<li>另外, 與 <a href="http://www.eva.mpg.de/lingua/resources/glossing-rules.php">Leipzig Glossing Rules</a> 之 Appendix 不同之處如下:
<ul>
<li><span class="sc">caus</span> 不用，標為 <span class="sc">cau</span>。</li>
<li><span class="sc">dem</span> 不用，標成 this, that。</li>
<li><span class="sc">nmlz</span> 不用，使用 <span class="sc">nmz</span>。</li>
<li><span class="sc">recp</span> 不用，使用 <span class="sc">rec</span>。</li>
</ul></li>
</ol>
</section>
<section id="篇章標記" class="level2">
<h2>篇章標記</h2>
<p>篇章標記大致依照DuBois的格式, 不同之處有：</p>
<ol type="1">
<li>Truncated intonation unit不用</li>
<li>為避免和 Leipzig Glossing Rules 衝突，Truncated word 從 - 改成 -<img src="newimages/spacer.gif" data-border="0" width="2" height="1" />-。</li>
<li>為避免和 Leipzig Glossing Rules 衝突，Lengthening 從 = 改成 ==。</li>
<li>‘^’ 符號不表 primary accent, 在此語料庫, ‘^’ 標語氣加重之音節。</li>
</ol>
